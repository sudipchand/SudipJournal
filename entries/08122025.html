CSS

CSS is also a thing I must learn for a webpage. So till now I know these:

  • css can reset browser defaults, margin=0
  • set global styles for the entire page: body { font-family: Arial; background-color: #f7f7f7; color: #222; }
  • navigation bars are styled: nav { background-color: #333; padding: 10px; }
  • style links: nav a { color: white; text-decoration: none; }
  • :hover pseudo-class: nav a:hover { color: #ffd700; } This changes styles when the mouse moves over the element.
  • .class selector works: .container { max-width: 800px; margin: auto; padding: 20px; } It is applied to elements using class="..." and can be reused again.
  • CSS can control layout width: max-width: 800px;
  • centers an element horizontally: margin: auto;
  • CSS can set background color for specific sections.
  • CSS can change fonts of specific elements: h1, h2 { font-family: "Georgia", serif; }
