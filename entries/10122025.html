Aluminium nanowire structure simulation

In the afternoon when I was working on the Al nanowire, I found the actual problem that was bothering me for so many days. And the important step in solving a problem is understanding the problem. Then we had to cut or shape the rectangular block like a hexagonal nanowire using awk which worked without any issue.

The next step was to relax the microstructure so that there are no residual forces. It usually means to minimize the total potential energy so that the particles arrange themselves. The relaxed structure now has to go through thermal equilibrium. For now I haven't gone through what the thermal equilibration file does or the method but I will do it definitely after I prepare the .ldata file for the simulation.

In the relaxed .ldata file the atoms are going outside the simulation box so when I ran the simulation it says the atoms are not numbered properly. This is happening because the atoms outside the sim box are not considered. So, now I have to figure out a way to tackle this.
